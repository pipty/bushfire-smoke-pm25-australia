# Historical Bushfire-Smoke PM2.5 Exposure Reconstruction for Australia

This repository contains a public-facing GitHub Pages site for a Centre for Safe Air Postdoctoral Fellowship project led by Dr Tingting Ye.

The public site intentionally presents project aims, conceptual framework, milestones, and planned outputs. Detailed data inventories, model diagnostics, preliminary results, and 6-month review evidence should remain private until they are ready for broader release.

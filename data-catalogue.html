Current data source and modelling-input catalogue

This page is the durable home for the project data inventory. It records public-facing descriptions of the modern reference outcome and currently available historical predictor datasets without exposing internal paths, raw files, or restricted intermediate products.

Latest source update

Historical predictor stack assembled for monthly reconstruction

The July 2026 progress update defines the current predictor stack and staged modelling strategy. The primary modelling target is monthly bushfire-smoke PM2.5 reconstruction from 1960 onward, calibrated against the modern CARE reference period.

The dated progress page remains an evidence record; this catalogue is the maintained summary of the datasets currently available for modelling.

Project Framework

Staged reconstruction logic and confidence classes

The fellowship separates modern benchmark evidence, historical extension, and exploratory reconstruction so outputs can be interpreted with period-specific confidence.

Work package 1

Characterise the CARE modern reference period and derive exposure metrics useful for epidemiological and climate-health applications.

Work package 2

Assemble harmonised fire, meteorological, land-cover, topographic, and socio-demographic predictors for historical extension.

Work package 3

Develop and test reconstruction models with uncertainty communication tailored to each historical period.
